ScanicDocument scanning for the modern Web

Professional-grade edge detection and perspective correction in the browser and Node.js — powered by Rust/WebAssembly. Under 100KB, ~10ms transforms.

⚡

Near-native speed

Rust/WASM handles the CPU-heavy edge detection; a GPU-accelerated triangle-subdivision warp does perspective correction in under 10ms.

📦

Featherweight

Under 100KB gzipped — versus 30MB+ for OpenCV.js. The WASM is inlined, so there are no extra files to host or fetch.

🎯

Accurate detection

Robust document contour detection with an adaptive multi-pass cascade that holds up even in low-contrast, noisy photos.

🛠️

Modern API

Clean, promise-based API with full TypeScript support. One call to detect, one call to extract.

🌐

Runs everywhere

Browser, Node.js, and Electron. A pure-JS fallback keeps it working even where WASM can't run.

🤚

Manual corner editor

A built-in, touch-friendly corner editor lets users fine-tune detected corners before extraction.

Quick start ​

sh
npm install scanic
sh
pnpm add scanic
sh
yarn add scanic
sh
bun add scanic
js
import { scanDocument } from 'scanic'

// Detect the document and get a perspective-corrected crop
const result = await scanDocument(imageElement, { mode: 'extract' })

if (result.success) {
  document.body.appendChild(result.output) // a <canvas> with the flattened page
}
